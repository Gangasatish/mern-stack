# HTTP Status

## 1xx - informational
- 100 - Continue
## 2xx - Success
- 200 - OK
- 201 - Created
- 204 - No Content/success but not response body
## 3xx - Redirection
- 301 - Permanent redirection
- 302 - Temporary redirection
## 4xx - Client Errors
- 400 - Bad request
- 401 - Unauthorized
- 403 - Forbidden
- 404 - Not found
- 409 - Conflict
- 422 - Validation error
## 5xx Server Errors
- 500 - Internal Server Errors
- 502 - Bad gateway
- 503 - Service not available
