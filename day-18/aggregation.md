# What is Aggregation?
- Aggregation processes multiple documents and tranform them into a summarized or computerized results.

## Benifits of Aggregation:
- Group data
- calculate totals
- Filter and reshape data
- join collection
- create reports

## Operators
- Comparision Operators($eq, $ne, $gt, $gte, $in, $nin)
- Logical operators($and, $or, $not, $nor)
- Element operators($exists, $type)
- Update operators($set, $unset, $push, $pop, $inc)
- Array Operators($size)
- Aggregate Operators($sum, $avg, $min, $max)
- Evaluation Operators($regex, $expr, $mod, $text)
